# Install
+ You can  simply run install.sh or do it manually below

+ Install requirements
```bash
pip install requests pytelegrambotapi
```
+ Clone repository
```bash
git clone https://github.com/mlambotaku9/FBI-Wantedbot && cd FBI-Wantedbot && chmod +x install.sh
```
+ Run the bot
```bash
python3 main.py
```
